段落和句子的作用

  • 开头：

    • 内容：写了 ... + 深层感情'!'

    • 结构：总领全文、引出下文、作铺垫、埋伏笔、设置悬念、激发读者阅读兴趣。

  • 中间：

    • 内容：承接上文 ... 开启下文 ... / 引出了 ...

    • 结构：承上启下、起过渡作用 / 引出下文 / 设置悬念、激发读者阅读兴趣 / 作铺垫 / 埋伏笔

  • 结尾：

    • 内容：写了 ... 总结概括文章大意、联系上文、分析深层含义

    • 结构：首尾呼应、使文章结构更严谨、总结全文、深化主题、点明中心

赏析词语

  • 答题模板：
    1. 解释词语的本意 + 联系上下文解释原因（为什么要这样用）（多为动词）
    2. 解释词语的本意 + 解释词语在文中的含义 + 联系上下文解释原因（为什么要这样用）（多为形容词副词一类）

赏析句子

  • 修辞手法：
    1. 比喻：把……比作……，生动形象地表达了（体现了）……和作者的……
    2. 拟人：把……人格化，生动形象地表达了（体现了）……和作者的……
    3. 排比：把……并列，句式整齐，节奏鲜明，表达了（体现了）……
    4. 对比：把…和…对比，突出强调了…的……，表达了（体现了）……
    5. 反复：强调了……，表达了（体现了）……的作者的……
  • 描写手法：
    1. 社会/自然环境描写：
      • 答题模板：描写了…… + 环境描写的作用
      • 描写作用：交代故事发生的时间，地点，烘托人物心情 + 奠定文章感情基调 + 渲染气氛
    2. 外貌/心理/动作/语言：描写了……结合文章谈作用

标题的好处

  • 新颖，设置悬念，吸引读者阅读兴趣
  • 解释含义：表层&深层
  • 情节围绕标题内容展开（线索）
  • 点明主旨

线索

  • 明线：由人物活动或事件发展直接呈现出来的故事线索
  • 暗线：通过直接描绘的人物活动或时间所间接呈现出来的，一般是思想感情一类的线索

议论文-开头

  • 引出文章的论题
  • 吸引读者（引发读者思考、激发读者阅读兴趣）
  • 作为 事实论据/道理论据

议论文语言特点

  1. 解释特殊词语的含义
  2. 总结一下句子的意思
  3. 体现出议论文语言准确严密的特点

'我'在小说中的作用

  • 讲述故事真实可信
  • 推动情节发展（线索）

第二人称作用

  • 直接对话，亲切自然
  • 直面对象，便于抒情
  • 对象为物则起到拟人作用

第三人称作用

  • 客观冷静
  • 叙述自由

说明文 阅读

说明对象

  • 位置：一般都在标题和首段里
    1. 设问方式：
      1. 说明对象下定义：。。。是。。。（找到主谓宾，再添定状）
      2. 说明对象含义：1）按点给分；2）首段

说明语言

  • 设问方式：

    1. “能不能删？”

       回答："不能删"+删去前文章的意思+删去后文章的意思+"所以删去后文中的意思与原意不符，所以不能删。这体现了说明文语言的准确性、严谨性、科学性。"

    2. “能不能换？”

       回答："不能换"+换去前文章的意思+换去后文章的意思+"所以换去后文中的意思与原意不符，所以不能换。这体现了说明文语言的准确性、严谨性、科学性。"

    3. “加点词表达效果？”

       回答：词语的本意+句子的意思+"有了这个词语，句子表达效果能更准确"

说明顺序

 时间、空间、逻辑（非空间、非时间）

说明方法

  1. 举例子：举了。。。的例子，形象具体地说明了。。。的。。。
  2. 列数字：列举了。。。的数据，准确具体的说明了。。。的。。。
  3. 作比较：把。。。和。。。作比较，突出强调了。。。的。。。
  4. 打比方：把。。。比作。。。，生动形象地写了。。。的。。。
  5. 下定义：科学简明地揭示了。。。的本质 判断：A是B，B也是A
  6. 做诠释：通俗易懂的阐明了。。。的意义 判断：A是B，B不是A
  7. 分类别：把。。。分门别类，使说明更有条理性。
  8. 引资料：说明了。。。的事例，强调了。。。的。。。提高文章准确性，使文章更具说服力
  9. 列图标：加入了表明。。。的。。。的图表，使文章更具说服力
  10. 摹状貌：对。。。的外貌进行了描写，体现了。。。的。。。；吸引读者/激发读者阅读兴趣/引发读者思考，使文章更具说服力

说明文-段落/句子作用

  • 开头
    1. 引出说明对象（具体）
    2. 引出下文对说明对象的。。。
    3. 设置悬念，吸引读者/激发读者阅读兴趣/引发读者思考
  • 中间
    1. 承上启下/启下/引出下文
  • 结尾
    1. 总结段落大意
    2. 知识延申

作文

  • 要点1：
    1. 选材
    2. 列提纲：
      • 中心：通过叙述……事情，抒发了……感情/明白了……道理
      • 详细：切片式描写（几乎能用手法的地方就用手法）（尽量不少于5段）
        1. 第一段 写……（手法）（大概内容）
        2. ......
        3. 第n段 过渡（手法）（大概内容）
        4. 第n段 结尾（手法）（大概内容）
