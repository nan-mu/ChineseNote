记叙文

段落/句子作用

  • 开头：

    • 内容：写了 ... + 深层感情

    • 结构：总领全文、引出下文、作铺垫、埋伏笔、设置悬念、激发读者阅读兴趣。

  • 中间：

    • 内容：承接上文……开启下文……/ 引出了……

    • 结构：承上启下、起过渡作用 / 引出下文 / 设置悬念、激发读者阅读兴趣 / 作铺垫 / 埋伏笔

  • 结尾：

    • 内容：写了 ... 总结概括文章大意、联系上文、分析深层含义

    • 结构：首尾呼应、使文章结构更严谨、总结全文、深化主题、点明中心

赏析词语

  • 答题模板：

    1. 解释词语的本意 + 联系上下文解释原因（为什么要这样用）（多为动词）

    2. 解释词语的本意 + 解释词语在文中的含义 + 联系上下文解释原因（为什么要这样用）（多为形容词副词一类）

赏析句子

  • 修辞手法：

    1. 比喻：把…比作……，生动形象地表达了（体现了）……和作者的……

    2. 拟人：把……人格化，生动形象地表达了（体现了）……和作者的……

    3. 排比：把……并列，句式整齐，节奏鲜明，表达了（体现了）……

    4. 对比：把…和…对比，突出强调了…的……，表达了（体现了）……

    5. 反复：强调了……，表达了（体现了）……的作者的……

  • 描写手法：
    1. 社会/自然环境描写：

      • 答题模板：描写了…… + 环境描写的作用

      • 描写作用：交代故事发生的时间，地点，烘托人物心情 + 奠定文章感情基调 + 渲染气氛

    2. 外貌/心理/动作/语言：描写了……结合文章谈作用

标题

  • 新颖，设置悬念，吸引读者阅读兴趣

  • 解释含义：表层&深层

  • 情节围绕标题内容展开

  • 点明主旨

线索

  • 明线：由人物活动或事件发展直接呈现出来的故事线索

  • 暗线：通过直接描绘的人物活动或时间所间接呈现出来的，一般是思想感情一类的线索
